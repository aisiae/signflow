교육 명단 관리 시스템

교육 정보와 참여자 명단을 등록하면 메인 페이지에서 바로 사용할 수 있습니다.

차수나 반 정보 등을 입력할 수 있습니다
콤마(,)로 구분해서 입력하거나 한 줄에 한 명씩 입력해주세요
메인 페이지 URL이 여기에 표시됩니다
명단 등록 후 위 주소를 참여자들에게 공유하세요
📋 사용 방법
1. 교육 정보와 참여자 명단을 입력하고 "명단 등록하기" 클릭
2. 메인 주소를 참여자들에게 공유
3. 참여자들이 메인 페이지에서 본인 이름을 클릭하여 서명 진행
4. 필요시 "일괄 PDF 생성"으로 모든 서명을 한 번에 다운로드
✨ 주요 특징
• 개별 링크 생성 불필요 - 하나의 주소로 모든 참여자 접근
• 실시간 명단 업데이트 - 새로운 명단 등록시 즉시 반영
• 자동 정보 입력 - 참여자가 이름만 클릭하면 교육 정보 자동 입력
• 모바일 최적화 - 스마트폰에서도 편리한 서명 기능
